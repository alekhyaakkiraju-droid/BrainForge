# BrainForge Sound Assets

Place sound effect files in this directory.

## Required files

| File | Description |
|------|-------------|
| `start_ritual.mp3` | 1–2 second power-up chime played during the Start Ritual animation. Replace this placeholder with a royalty-free sound effect before release. |

## Licensing
